Solving Ordinary Differential Equation (ODEs):

ODEs: Single variable equations that involve an arbitrary amount of derivatives of that function wrt that variable.

ODE System: System of ODEs. Linear or non linear.

Problem w non linear equations: harder. Linear eqs are simpler given boundary conditions.

Classical computers can only use non continuous data. So one can only estimate solutions of hard ODEs in a 1D region [t0, tf] by obtaining their images given a mesh {ti} = i(tf-t_0)/n, where n is a natural number greater than 0 that we can choose. Bigger n implies finer mesh but longer computation time (usualy of order O(n)O(n)O(n)).
