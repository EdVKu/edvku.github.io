Jacobi Algorithm for Real Symmetric Matrices

  1. At each step find the largest off diagonal element, say SpqS_{pq}Spq​ (or cycle thru each off-diagonal index p,qp,qp,q one at a time)

  2. Transform it to an orthogonal matrix JJJ s.t. that element and its symmetric version are both 0

  3. Check whether the relative error δ<ϵ\delta <\epsilonδ<ϵ where ϵ\epsilonϵ is the desired error. If not, repeat the previous steps
